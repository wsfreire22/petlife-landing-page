# 🐾 PetLife - Landing Page de Alta Conversão

> Link do projeto online: [Insira aqui o link do deploy na Vercel ou Netlify]

Este projeto é uma Landing Page responsiva e de alta performance, desenvolvida com o objetivo de converter usuários e capturar leads para o lançamento de um guia digital sobre bem-estar e cuidados com animais de estimação. 

O projeto foi construído do zero seguindo a filosofia de **MVP (Minimum Viable Product)**, focando em uma entrega rápida, código extremamente limpo e na resolução do problema principal de negócio: a captura de dados de potenciais clientes.

---

## 🎯 Funcionalidades e Escopo do MVP

Para garantir um desenvolvimento ágil focado em conversão, o escopo foi reduzido às seções essenciais de uma página de vendas:
*   **Dobra Principal (Hero Section):** Proposta de valor clara, imagem otimizada e chamada para ação (CTA) imediata.
*   **Seções de Benefícios:** Apresentação dos pilares do produto focado em escaneabilidade visual.
*   **Formulário de Captura Inteligente:** Validação de campos em tempo real com JavaScript (garantindo e-mails válidos).
*   **Simulação de Persistência:** Interceptação do envio do formulário para exibir uma mensagem de sucesso dinâmica ao usuário, salvando temporariamente o lead no `localStorage` do navegador para simular o comportamento de um banco de dados.

## 🛠️ Decisões Técnicas & Aprendizados

Como estudante de Desenvolvimento Web, este projeto foi essencial para consolidar conceitos fundamentais do ecossistema front-end:
*   **Desenvolvimento Mobile-First:** A página foi estruturada pensando primeiro na experiência em smartphones (onde ocorre a maior parte dos acessos atuais) e adaptada progressivamente para desktop usando CSS Flexbox e Media Queries.
*   **HTML5 Semântico:** Uso rigoroso de tags como `<header>`, `<main>`, `<section>` e `<footer>` para garantir uma boa pontuação de SEO e acessibilidade (critérios fundamentais para páginas de conversão de mercado).
*   **Manipulação de DOM nativa:** Validação e lógica do formulário implementadas com JavaScript Vanilla, sem a dependência de frameworks externos, para garantir o domínio sobre o comportamento assíncrono e eventos da web.

## 🚀 Tecnologias Utilizadas

*   **HTML5** - Estruturação semântica.
*   **CSS3** - Estilização moderna, responsividade e variáveis nativas.
*   **JavaScript (ES6+)** - Lógica de validação e manipulação de eventos.
